# P360

> Real-time Bio-data Driven Decision Support for Biohackers & High-performers

## What & How

**What we do** — P360는 앱이 아니라 **함수**입니다:
```
p360(pain) → {solution₁, solution₂, solution₃, ...}
```

**How we execute** — 모든 작업은 이 파이프라인을 따릅니다:
```
p(x) → s(y) → tool-first GTM → do things that don't scale → make money
```

## Structure

```
p360/
├── apps/                    # 클라이언트 앱들
│   ├── cli/                 # CLI 도구
│   ├── raycast/             # Raycast 익스텐션
│   └── telegram/            # Telegram 봇
├── packages/                # 공유 패키지
│   └── core/                # 코어 로직
└── docs/                    # 문서
    ├── core/                # 핵심 문서 (P360_CORE.md 등)
    ├── legal/               # 법률 문서
    ├── research/            # 리서치
    ├── gtm/                 # GTM 자료
    └── archive/             # 구버전 참고용
```
