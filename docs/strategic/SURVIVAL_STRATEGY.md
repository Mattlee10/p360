## Daily Routine (08:55 시작)

### 09:00-12:00: Type_A(Signal 3) 찾기 + 1st DM
**목표:** 매일 1명 DM 발송 (채널별 1명)

#### 1. Reddit 스캔 (30분)
- r/ouraring (HOT posts)
- r/whoop (HOT posts)
- r/dexcom (HOT posts)

#### 2. Pain Point 매칭 (30분)
**ICP_TYPEA.md 체크리스트:**
- Signal 3 (Decision Pain): ≥40% 필터링
- Signal 1 + 2 + 4 (Confounding, Pattern, Numerical): ≥60% 합산
- **CRITICAL:** Signal 3 <40% → SKIP

#### 3. Humanizer 검토 (15분)
- 1st DM 작성 후 `/humanizer` skill 사용
- 제거: em dash, promotional language, vague attribution, AI vocabulary
- 목표: "사람이 쓴 것처럼"

#### 4. DM 발송 (15분)
**구조:**
```
1st DM = 공감 + 관찰 + 질문

"i saw your post about [pain].
your data shows [observation]...
want me to analyze?"
```

   - API 에러 디버깅
